# Guia De Diseno

## Concepto

Modern Minimal Sacred UI.

Una experiencia moderna, calmada y elegante enfocada en lectura, estudio, concentracion y proyeccion.

## Referencias Locales

Las referencias visuales estan en la carpeta `estilos`.

### `estilos/estilo 1.jpg`

Referencia general de la plataforma:

- Tema oscuro suave.
- Sidebar vertical minimalista.
- Panel principal organizado.
- Tarjetas y secciones con aire.
- Sensacion premium y moderna.

### `estilos/estilo de lectura.jpg`

Referencia para una experiencia de lectura clara:

- Fondo claro calido.
- Panel lateral limpio.
- Lectura amplia y tranquila.
- Navegacion inferior tipo paginas.
- Buena separacion entre contenido y controles.

### `estilos/estilo de lectura 2.jpg`

Referencia para lectura tipo doble pagina:

- Composicion editorial.
- Buscador superior.
- Sidebar funcional.
- Barra inferior con controles.
- Lectura dividida en columnas o paginas.

Estas referencias son direccion visual, no restricciones absolutas.

## Sensacion Deseada

La aplicacion debe sentirse:

- Calmada.
- Clara.
- Elegante.
- Ligera.
- Moderna.
- Profesional.
- Comoda para sesiones largas.

## Tipografia

### Texto Biblico

Usar serif elegante:

- Merriweather.
- Crimson Pro.
- Literata.
- Source Serif.
- EB Garamond.

### UI

Usar sans moderna:

- Inter.
- Manrope.
- Geist.
- Estilo SF Pro.

## Temas

### Modo Claro

- Blanco roto.
- Fondos calidos suaves.
- Sensacion papel moderno.
- Contraste suficiente para lectura prolongada.

### Modo Oscuro

No usar negro puro. Preferir:

- `#111827`
- `#0F172A`
- `#1C1C1E`

## Paleta

Base:

- Blanco suave.
- Gris calido.
- Negro suave.

Acentos:

- Azul elegante.
- Dorado sutil.
- Ambar suave.
- Violeta profundo.

Evitar:

- Colores saturados.
- Estetica religiosa exagerada.
- Interfaces recargadas.

## Componentes

- Sidebar colapsable.
- Buscador rapido.
- Selector de versiones.
- Panel de lectura.
- Barra inferior contextual.
- Panel de notas separado.
- Vista de proyeccion limpia.

## Animaciones

Usar:

- Fade suave.
- Slide horizontal.
- Transicion tipo pagina.
- Blur ligero cuando aporte claridad.

Evitar:

- Rebotes exagerados.
- Efectos gamer.
- Transiciones pesadas.
